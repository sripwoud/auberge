# Development Setup

## Quick Start

```bash
./setup              # Install tools (mise, Rust, Ansible)
cargo build          # Build
cargo test           # Test
```

## Tools Installed

- mise - Environment manager
- Rust toolchain
- Ansible and ansible-lint
- dprint, dasel, pkl

## Commands

```bash
cargo build --release    # Build release binary
cargo test               # Run tests
cargo clippy            # Lint
ansible-lint            # Lint playbooks
```
